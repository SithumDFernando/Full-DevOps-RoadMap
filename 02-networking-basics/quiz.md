# Quiz 02

1. Which port is typically used for HTTPS?
<details><summary>Answer</summary>
Port 443.
</details>

2. What is the difference between TCP and UDP?
<details><summary>Answer</summary>
TCP is reliable and ensures packets arrive in order (e.g., web traffic, SSH). UDP is fast but does not guarantee delivery or order (e.g., video streaming, gaming).
</details>

3. Which DNS record type maps a domain name to an IPv4 address?
<details><summary>Answer</summary>
A Record.
</details>

4. If a web server returns a `404` status code, what does it mean?
<details><summary>Answer</summary>
Not Found - The client was able to communicate with the server, but the server could not find what was requested.
</details>

5. What happens if you share your SSH private key?
<details><summary>Answer</summary>
Anyone with your private key can impersonate you and access any servers or services (like GitHub) that trust that key. Never share it!
</details>
