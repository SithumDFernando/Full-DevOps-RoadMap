# Exercises 01

1. Write a script that loops through the numbers 1 to 5 and prints them.
2. View your environment variables. Export a variable named `LAB_ENV` with the value `testing`. Write a command that prints it.
3. Write a script `cleanup-logs.sh` that finds all `.log` files in a given directory older than 7 days and deletes them.
   - **Requirement:** Add a `--dry-run` flag so it only prints what it *would* delete without actually deleting anything.
   - *Hint:* Use the `find` command.
