# Module 00: Prerequisites

Welcome to the DevOps Learning Path! Before we dive into the technical content, we need to ensure your machine is set up correctly and you have the necessary accounts.

## 🛠️ Required Tools

You will need the following tools installed on your local machine:

1. **Terminal Emulator**:
   - macOS: iTerm2 or the built-in Terminal app.
   - Windows: Windows Terminal with WSL2 (Windows Subsystem for Linux).
   - Linux: Your favorite terminal (GNOME Terminal, Kitty, Alacritty, etc.).
2. **Git**: Version control system.
3. **Code Editor**: VS Code (Visual Studio Code) is highly recommended.
4. **Docker Desktop**: For running containers locally.

## ☁️ Required Accounts

1. **GitHub**: Create a free account at [github.com](https://github.com).
2. **Cloud Provider (AWS/Azure/GCP)**: We will use AWS as our primary reference. Create a free-tier account at [aws.amazon.com](https://aws.amazon.com).

## 📂 Repo Structure Walkthrough

This repository is organized into sequential modules (`00-prerequisites`, `01-linux-and-shell`, etc.). Each module contains:
- `README.md`: Core concepts, theory, and diagrams.
- `lab.md`: Step-by-step, hands-on exercises. Don't skip these!
- `exercises.md`: Independent practice tasks.
- `quiz.md`: A short knowledge check before moving to the next module.
- `scripts/` (optional): Helper scripts used in labs.

## 📝 How to use the Lab & Quiz format

1. Read the `README.md` to grasp the concepts.
2. Follow the `lab.md` strictly. Use the provided copy-pasteable blocks to get familiar with the commands.
3. Verify your work against the "Expected Result" blocks.
4. Test yourself with `exercises.md` and `quiz.md`.
