# Exercises 00

1. **Fork this repository:** Go to the top right of this GitHub repo and click "Fork" to create your own copy.
2. **Clone your fork locally:**
   ```bash
   git clone git@github.com:<your-username>/Full-DevOps-RoadMap.git
   cd Full-DevOps-RoadMap
   ```
3. **Create a `LEARNERS.md` file:** Add your name to the file.
   ```bash
   echo "- Your Name" > LEARNERS.md
   ```
4. **Make your first commit and push:**
   ```bash
   git add LEARNERS.md
   git commit -m "docs: add my name to LEARNERS.md"
   git push origin main
   ```
