# Quiz 00

1. What is a terminal emulator?
<details><summary>Answer</summary>
A terminal emulator is a program that allows you to interact with the shell (command-line interface) of your operating system.
</details>

2. What is the difference between a fork and a clone?
<details><summary>Answer</summary>
A fork is a copy of a repository on GitHub (server-side), while a clone is a local copy of a repository on your computer.
</details>

3. What is the difference between Git and GitHub?
<details><summary>Answer</summary>
Git is the local version control software you install on your machine. GitHub is a cloud-based hosting service for Git repositories.
</details>

4. What is the purpose of a `README.md` file?
<details><summary>Answer</summary>
A README file is typically the first document a user reads when encountering a new project. It provides instructions, documentation, and orientation.
</details>

5. Why do we generate an SSH key for GitHub?
<details><summary>Answer</summary>
An SSH key allows you to securely authenticate with GitHub without having to type your username and password every time you push or pull code.
</details>
